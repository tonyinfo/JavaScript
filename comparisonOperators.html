Comparison Operators

Often used in Boolean to compare whether an operands is true or false If the operands are numbers, the numeric comparison will be performed but if they are strings, the Script will evaluate them and perform comparison.

Operators Description == Equality === Identical > Greater than < Less than >= Greater or equal <= Less or equal != Not equal

Compering numeric values

let a = 5, b = 10; console.log(a >= b); // false console.log(a == 10); // false Five is not greater and is not equal to ten.

Compering Strings

When the operands are strings, the Script will evaluate the strings and perform the appropriate comparison.

let f = 'Favour', d = 'Divine'; let result = f < d; console.log(result); // false console.log(f == d); // false console.log(f > d); // true console.log(f == 'Favour' ); // true console.log(d == 'Favour'); // false console.log(d == 'Divine'); // true

p>The answer is as it because the ASCII character code for ‘f’ is 070 and ‘d’ is 068, with binary codes of 01000110 and 01000100 respectively.

Compering numbers

//Compering numbers let a = 10; //number b = '10'; //string console.log(a == b); // true console.log(a === b); //false console.log(a !== b);//true console.log(a != b); //false Document
